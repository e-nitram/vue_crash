| Attribute   | Description         | Type      | Accepted values | Default |
| ----------- | ------------------- | --------- | --------------- | ------- |
| **width**   | content width src   | `string`  | -               | `auto`  |
| **caption** | caption for display | `string`  | `template`      | -       | - |
| **shadow**  | content with shadow | `boolean` | -               | `false` |
