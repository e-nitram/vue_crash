## 介绍

### 什么是 Geist UI - VUE

`@geist-ui/vue` 是针对 <zi-link pure href="https://github.com/geist-org/style">Geist 样式库</zi-link> 的一个 Vue 版本实现。

**Geist Org** 是非官方的组织，且所有项目都是开源的。
