| 属性          | 描述                   | 类型                      | 推荐值 | 默认    |
| ------------- | ---------------------- | ------------------------- | ------ | ------- |
| **value**     | 展开操作的值           | `array | string | number` | -      | -       |
| **accordion** | 手风琴模式             | `string | boolean`        | -      | `false` |
| **title**     | 用于显示的标题         | `string`                  | -      | -       |
| **name**      | 触发事件用于标识的字符 | `string | number`         | -      | -       |
| **shadow**    | 阴影效果               | `boolean`                 | -      | -       |
| **subtitle**  | 标题详情               | `string`                  | -      | -       |
